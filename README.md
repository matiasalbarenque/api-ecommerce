# UADE - Aplicaciones Interactivas 
## Proyecto: Ecommerce (Front-End)
### Tecnologías: React + Vite + Tailwind CSS + Ant-Design

---

## Instalación de dependencias:

```bash
npm install

```

## Iniciar servidor de data mockup

```bash
npm run json-server
```

## Iniciar server de desarrollo

```bash
npm run dev
```

## Compilación

```bash
npm run build
```
